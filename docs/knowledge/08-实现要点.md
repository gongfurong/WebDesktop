# 实现要点

## 最小能力

- `manifest.json`
- `service worker`
- 应用图标
- `display: standalone`
- 安装引导 UI

## 页面引导建议

- 桌面端：优先突出浏览器原生安装入口
- iOS / iPadOS：优先突出“添加到主屏幕”
- Android：优先突出原生安装，次选浏览器菜单安装

## 示例代码应该承担什么

- 提供可运行、可测试、可观察的参考实现
- 展示关键平台差异与主路径
- 不要求每次实现都复制全部分支

## 示例代码不应该承担什么

- 不应替代完整知识库
- 不应被视为所有项目的默认最终形态
- 不应强迫所有平台走同一条实现路径
