# 网页桌面化知识总览

## 目标

这一组文档是共享知识库，面向人类和 AI 共同使用。

特点：

- 所有知识点独立成模块
- 术语统一、便于查阅
- 适合按需检索，不要求一次读完

## 推荐阅读顺序

### 人类读者

1. `01-术语表.md`
2. `02-平台差异.md`
3. `03-浏览器差异.md`
4. `05-安装路径.md`
5. 按需查看 `06-证书与信任.md`、`07-验收与清理.md`

### AI / 开发执行

1. 不要先读完整个知识库
2. 先读 `../ai/00-入口路由.md`
3. 再按路由要求回到这里读相关模块

## 模块说明

- `01-术语表.md`
  - 统一术语定义
- `02-平台差异.md`
  - Windows、macOS、Linux、iOS / iPadOS、Android 差异
- `03-浏览器差异.md`
  - Chrome、Edge、Brave、Opera、Safari、Firefox 差异
- `04-安全与协议.md`
  - `localhost`、网络地址、可信 HTTPS、不可信 HTTPS 的区别
- `05-安装路径.md`
  - 各平台推荐安装路径
- `06-证书与信任.md`
  - 证书分发、导入、信任、删除
- `07-验收与清理.md`
  - 安装验收、卸载、证书清理
- `08-实现要点.md`
  - `manifest`、`service worker`、图标、安装 UI 等最小实现要点
- `09-证书、签发与站点信任.md`
  - 证书、签发、站点证书、私钥、CSR、SAN、OpenSSL 等基础概念
- `10-示例项目说明与测试.md`
  - 说明示例项目的定位、阅读顺序、启动停止方式、测试方式，以及 AI 与人类如何参考它

## 使用原则

- 这里放知识，不放执行工作流
- 这里放事实与差异，不放默认决策
- AI 的默认策略、询问规则、输出结构，统一放在 `../ai/`
