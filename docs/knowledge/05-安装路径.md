# 安装路径

## 桌面端（Windows / macOS / Linux）

主路径：

- 浏览器原生安装入口
- 常见按钮文案：`安装应用`、`安装桌面应用`

备选路径：

- 浏览器菜单中的安装入口

不推荐路径：

- 在弱支持浏览器中强行把它当作主安装方式

## iOS / iPadOS

主路径：

- Safari -> 共享 -> 添加到主屏幕

备选路径：

- 继续访问后再添加到主屏幕

不推荐路径：

- 等待标准桌面式安装弹窗

## Android

主路径：

- 浏览器原生安装入口

备选路径：

- 浏览器菜单中的“添加到主屏幕”或“安装应用”

不推荐路径：

- 把本地证书安装当成默认产品路径
