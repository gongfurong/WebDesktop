# 浏览器差异

## 桌面端主路径

### Chrome

- 安装入口支持强
- 独立窗口支持强
- `service worker` 支持强

### Edge

- 与 Chrome 接近
- 企业环境中常见

## 桌面端备选

### Brave

- 大体可用
- 行为多接近 Chromium

### Opera

- 通常可用
- UI 与安装提示位置可能不同

### Safari（macOS）

- 可用部分能力
- 不建议作为唯一桌面安装验证路径

## 弱支持或不推荐主路径

### Firefox

- 更适合浏览与调试页面
- 不建议作为安装主路径

## 移动端

### iOS / iPadOS

- Safari 是主路径
- 其他浏览器底层仍受 WebKit 规则约束

### Android

- Chrome 是主路径
- Edge、Brave、Opera 可作为备选
- Firefox 不建议作为安装主路径
