# 文档导航

## 结构

- `knowledge/`
  - 共享知识库
  - 人类和 AI 都可以直接阅读
- `human/`
  - 面向人类操作、使用、测试、验收
- `ai/`
  - 面向 AI 的路由、默认策略、工作流、输出模板

## 推荐入口

### 人类读者

1. `knowledge/00-总览.md`
2. `human/用户使用指南.md`
3. 按需查阅 `knowledge/*`

### AI / 开发执行

1. `ai/00-入口路由.md`
2. 按路由去读相关 `knowledge/*`
3. 再读 `ai/01-06`

## 设计原则

- 知识库负责“知识点与事实”
- AI 文档负责“路由、策略、询问、执行与输出”
- 人类文档负责“操作、验收与清理”
