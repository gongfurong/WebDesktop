# 测试与验收清单

## 安装验收

- 页面可正常打开
- 当前平台能按预期完成安装
- 安装后能从桌面或主屏幕入口启动
- 独立窗口行为符合预期

## 平台验收

### 桌面端

- 推荐浏览器中能看到安装入口
- 安装后有桌面或系统应用入口

### iOS / iPadOS

- Safari 中能按“共享 -> 添加到主屏幕”完成安装
- 添加后能从主屏幕入口打开

### Android

- 推荐浏览器中能通过原生入口或菜单完成安装

## 安全验收

- `localhost`、网络地址、可信 HTTPS 三种场景行为与预期一致
- 可信证书场景下安全提示消失或明显减少

## 清理验收

- 桌面 / 主屏幕图标可正常移除
- 证书可按平台路径正常删除
- 删除后再次访问时，行为与未安装状态一致
